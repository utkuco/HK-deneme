# HK Ortak Proje

Bu repo, iki ayrı Claude sohbetinin ortak hafızası olarak kullanılır.

## Nasıl çalışır?

- **Utku** ve **Seran** ayrı ayrı Claude ile çalışır.
- Her ikisi de önemli kararları, güncellemeleri ve bağlamı bu repoya push eder.
- Yeni bir oturuma başlamadan önce `CONTEXT.md` dosyası okunur → Claude'a verilir.

## Dosyalar

| Dosya | Açıklama |
|-------|----------|
| `CONTEXT.md` | Ortak proje bağlamı — her iki taraf günceller |
| `SERAN_GUIDE.md` | Seran için Claude kullanım rehberi |
| `UTKU_GUIDE.md` | Utku için Claude kullanım rehberi |
